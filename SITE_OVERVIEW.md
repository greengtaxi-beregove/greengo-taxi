# GreenGo Taxi — Технічний опис сайту

## Загальна інформація

**GreenGo Taxi** — лендінг-сторінка для служби таксі в місті Берегово, Закарпатська область.  
Сайт є одностороньовим (single-page) і призначений для залучення клієнтів: швидкий виклик авто через форму або дзвінок.

## Структура сайту

### Мобільна версія (< 1024px)

П'ять послідовних секцій:

1. **Hero** — фонове фото, заголовок "Твоє місто. Твоє таксі.", бургер-меню з навігацією
2. **Stats** — скляна панель з двома показниками: 5хв (середня подача) і 40+ (електрокарів)
3. **Form** — форма зворотного зв'язку на темному фоні із glass-ефектом
4. **Routes** — список популярних маршрутів: Chizay, Бакта, Жайворонок, Лужанка
5. **Features** — 4 картки переваг: електрокари, знання міста, швидкість, цілодобовість

Знизу екрану — **sticky-кнопка** з номером телефону та анімацією пульсації, зафіксована при прокрутці.

### Десктопна версія (≥ 1024px)

Одна секція на весь екран (`100vh`):

- Фонове фото `cover-pc.png` (автомобіль праворуч)
- Лівосторонній градієнтний оверлей для читабельності тексту
- Прозорий хедер із логотипом і кнопкою виклику
- Контентна колонка ліворуч: заголовок зеленим кольором, логотип, підзаголовок
- Внизу по центру екрану: маршрути-теги + горизонтальна форма замовлення

## Форма замовлення та Telegram-бот

### Як це працює

1. Користувач вводить ім'я та номер телефону і натискає "Викликати авто"
2. Форма надсилає `POST`-запит на внутрішній API-ендпоінт `/api/callback`
3. Сервер (Next.js Route Handler) отримує дані і надсилає повідомлення в Telegram через Bot API
4. Оператор отримує сповіщення в Telegram і передзвонює клієнту

### Технічна реалізація

- **API роут:** `app/api/callback/route.ts`
- **Метод:** `POST https://api.telegram.org/bot{TOKEN}/sendMessage`
- **Змінні середовища** (зберігаються в `.env.local`, не потрапляють у git):
  - `TELEGRAM_TOKEN` — токен бота
  - `TELEGRAM_CHAT_ID` — ID чату оператора

### Формат повідомлення в Telegram

```
🚖 Нова заявка GreenGo

👤 Ім'я: [ім'я клієнта]
📞 Телефон: [номер телефону]
```

### Стани форми

- **Звичайний стан** — поля введення + кнопка
- **Надсилання** — кнопка заблокована, текст "Надсилаємо..."
- **Успіх** — повідомлення "Дякуємо! Передзвонимо за 60 секунд"

## Дизайн і анімації

- **Glass morphism** — скляні картки з `backdrop-filter: blur()`, напівпрозорі фони, білі верхні обводки (rim highlight)
- **Blob-анімації** — фонові радіальні градієнти, що повільно дрейфують (CSS keyframes, 18–28с цикл)
- **Scroll reveal** — секції плавно з'являються при прокрутці через `IntersectionObserver`
- **Stagger delay** — елементи всередині секцій з'являються по черзі (затримка 0.07–0.5с)
- **Pulse-анімація** — кнопка дзвінка внизу екрану пульсує, привертаючи увагу

## SEO-налаштування

### Базова оптимізація

- **Title tag:** "GreenGo Taxi — Таксі в Береговому | Електрокар за 5 хвилин"
- **Meta description:** 160 символів із CTA та номером телефону
- **Keywords:** 14 цільових ключових слів (UA + EN варіанти назви міста)
- **Canonical URL** на основний домен
- **Open Graph** теги для коректного відображення у соцмережах (Facebook, Telegram)
- **Twitter Card** `summary_large_image`
- **lang="uk"** атрибут на `<html>`

### Технічне SEO

- **sitemap.xml** — автогенерація через `app/sitemap.ts` (Next.js built-in)
- **robots.txt** — автогенерація через `app/robots.ts`
- **Preload** для критичних шрифтів (TTFirsNeue-Black, TTFirsNeue-Regular) — покращує LCP
- **next/image** — автоматична оптимізація зображень (WebP, lazy loading, розміри)
- **SEO-текстовий блок** (`sr-only`) — прихований від користувача блок із H2/H3, списками маршрутів і `<address>` для пошукових роботів

### Structured Data (JSON-LD)

Три схеми в `<head>`:

1. **WebSite** — базова інформація про сайт
2. **LocalBusiness + TaxiService** — назва, адреса, телефон, геокоординати Берегового (48.2044°N, 22.6444°E), зони обслуговування, години роботи 24/7
3. **FAQPage** — 6 питань і відповідей для Google Featured Snippets

## AEO — Оптимізація для AI-пошуку

AEO (Answer Engine Optimization) — оптимізація для того, щоб сайт цитувався в AI-пошуковиках (ChatGPT, Perplexity, Google AI Overviews, Gemini, Claude).

### Що реалізовано

- **`/llms.txt`** — структурований текстовий файл для AI-агентів із описом сервісу, фактами, маршрутами та контактами (аналог `robots.txt`, але для AI)
- **FAQPage schema** — 6 питань у форматі, зрозумілому для AI: конкретні факти, числа, назви
- **AI-боти явно дозволені в robots.txt:**
  - GPTBot, ChatGPT-User (OpenAI / ChatGPT)
  - PerplexityBot (Perplexity)
  - ClaudeBot, anthropic-ai (Anthropic / Claude)
  - Google-Extended (Google Gemini / AI Overviews)
  - Bingbot (Microsoft Copilot)
- **CCBot заблокований** — Common Crawl використовується лише для тренування моделей без цитування

## Деплой

Сайт підготовлений до розгортання на **Vercel** (рекомендовано для Next.js):

- `TELEGRAM_TOKEN` і `TELEGRAM_CHAT_ID` потрібно додати як Environment Variables у налаштуваннях проєкту на Vercel
- Після деплою: підключити домен, додати сайт у Google Search Console, надіслати sitemap
